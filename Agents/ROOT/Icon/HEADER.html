This directory contains icons and other images that are shared widely throughout the PIA. Icons that are application-specific should go into that application's home directory. Icons and images used in documentation (i.e. in /Doc) should be local to the /Doc subtree, since it is occasionally separated from the rest of the PIA. It is expected that the top-level site configuration file will make this directory available at the root level as /Icon/.

Note:
When checking in binary files such as PNG's, you need to use: 
       cvs add -kb foo.png
Logos
A name like ``logo24.png'' indicates a plain logo; ``pia24.png'' indicates a logo attached to the letters ``PIA''. The number in all cases is the height in pixels.
GIF's
GIF images have been removed from the PIA and replaced with PNG images to avoid patent licensing problems. Besides, PNG's are smaller. GIF's can still be found in /Doc/API/javadoc/images -- these came from Sun, and we're presuming that they're ok. It's worth noting, though, that they are no longer used by recent versions of Javadoc.
