Icons for Agents

This directory contains icons that are shared by many agents. Icons that are agent-specific should go into the Agent's home directory. Putting icons under Agency allows them to be overridden by the user. It also ensures that they will be available; the Agency agent is always running.

Note:
When checking in binary files such as PNG's, you need to use: 
       cvs add -kb foo.png
Logos
A name like ``logo24.png'' indicates a plain logo; ``pia24.png'' indicates a logo attached to the letters ``PIA''. The number in all cases is the height in pixels.
GIF's
GIF images have been removed from the PIA and replaced with PNG images, to avoid patent licensing problems. Besides, PNG's are smaller. GIF's can still be found in /Doc/API/javadoc/images -- these came from Sun, and we're presuming that they're ok.
