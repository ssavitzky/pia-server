Workflow application

This directory contains a workflow application which allows users to fill out and digitally sign forms. Each form has an associated route, and ordered list of people who should sign the form. Forms are automatically moved from the pending list for one user to another as appropriate.

See About for more information.

  • templates contains "blank" forms to be filled out.
  • users contains user lists and individual pending forms
  • forms contains filled out forms organized hierarchically by type and date
