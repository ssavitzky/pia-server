lib

[..] [files]

This directory contains the ``libraries'' which are required for ordinary operation of the PIA and the DPS's process command. In particular, it contains the fundamental tagsets. Unlike the PIA server tagsets in ../Tagsets, tagsets in this directory are not expected to be customized.

Note that lib is added to the PIA's CLASSPATH.

Annotated Contents

Directories

dps
Contains tagsets used only in the stand-alone DPS environment, including the tsdoc and tsstrip tagsets.
