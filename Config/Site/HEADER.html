PIA/Config/Site

[..] [files]

This directory contains alternate site configuration files for defining the PIA's ``website'' (resource tree).

Usage: 
pia -site ~/PIA/Config/Site/sample.xcf

Annotated Contents

XML Configuration Files

sample.xcf
A sample configuration illustrating most of the common constructs.

XML Configuration Include files

It is possible to put arbitrary XXML code into configuration files. Some care must be taken in order not to rely on constructs, such as subdirectory Resource objects, that have not yet been configured!

This is buggy at the moment, so these may not work, or even exist, at first.

extensions.xci
The PIA's standard file-extension map.

Conventions

The following filename extensions are used:

.xcf
XML Configuration File
.xci
XML Configuration Include file.
