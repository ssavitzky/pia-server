PIA / Config / Site

[..] [files]

This directory contains alternate site configuration files for defining the PIA's ``website'' (resource tree).

Usage: 
pia -site ~/PIA/Config/Site/sample.xcf

Annotated Contents

XML Configuration Files

sample.xcf
A sample configuration file illustrating most of the common constructs.
my-system.xcf
A sample configuration file that lets you view your computer's entire filesystem as if it was a website.

XML Configuration Include files

It is possible to put arbitrary XXML code into configuration files. Some care must be taken in order not to rely on constructs, such as subdirectory Resource objects, that have not yet been configured! There may be some bugs at first, but at least in its simplest form it seems to be working.

extensions.xci
The PIA's standard file-extension map.

Conventions

The following filename extensions are used:

.xcf
XML Configuration File
.xci
XML Configuration Include file.
