[..] [files] ( setuid tools )

This directory contains the source code for ``applications'' -- stand-alone programs used either as development tools or as components of an information appliance.

Annotated Contents

html2latex/
Convert HTML documents to LaTeX for printing.
setuid/
Programs intended to be installed with the "setuid" (set user id) bit set, so that an ordinary user can execute programs under some other user's (for example, pia's) permissions.
tools/
Short scripts and other ``hacks'' used during the development process.
