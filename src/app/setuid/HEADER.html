This directory contains the source code for applications that have to run under a specific user ID on Unix. Typically these are wrappers for either system utilities (e.g. shutdown) or PIA utilities (e.g. pia).

Annotated Contents

be_pia.c
Execute a command or start a shell under the pia user ID.
run_pia.c
Run the PIA under the pia user ID.
shutdown.c
Shut down the computer (shutdown -h).
