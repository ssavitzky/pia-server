Java Makefiles

[..] [files]

This directory contains makefile templates; they are meant to be included using the ``include'' directive of GNU or similar versions of make. They are known to work on Linux with GNU make and Solaris with SUN's make.

A little attention has been paid to making it work on Windows, but unless make, java, and the shell interpret ``/'' as the file separator there is little hope that they will really work.

Annotated Contents

file.make
Template for making Java classes from .java files.
package.make
Template for a directory that contains sub-packages.
