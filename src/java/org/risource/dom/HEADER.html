DOM Extension for Active Documents

[..] [files]

This directory contains an extension of the Document Object Model (DOM: org.w3c.dom) for ``active documents'' as used in the PIA and its Document Processing System.

Originally this package simply contained a complete implementation of (an earlier version of) the DOM, and dps.active extended it. Now that the ``official'' interfaces are in org.w3c.dom we have taken the opportunity to correct the situation.

Interfaces

Interfaces in this package have names that start with ``Active''. In general, they are also implementations of org.risource.dps.Active and its extension ActiveNode.

Classes

org.risource.dom.tree
contains classes that implement the Active DOM interfaces in this package using a simple doubly-linked tree representation.

Eventually it is possible that we will have multiple implementations. Some of the obvious possibilities are a threaded tree (to make the dynamic nodelists returned by getElementsByTagName efficient)
