Document Object Models

[..] [files]

This package, org.risource.dom, is meant to contain utility classes that use the ``pure'' DOM interfaces from org.w3c.dom.

Originally this package simply contained a complete implementation of (an earlier version of) the DOM, and dps.active extended it. Now that the ``official'' interfaces are in org.w3c.dom we have taken the opportunity to correct the situation.

See Also

org.risource.dps.active
Interfaces that combine the DOM and Active interfaces. Classes that implement these interfaces can be efficiently traversed using cursors.
org.risource.dps.tree
Classes that implement the DOM and Active interfaces, for use in building parse trees.
