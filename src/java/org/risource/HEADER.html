Naming Conventions

Every effort has been made to reduce directory depth in order to accomodate Microsoft and ISO9660 restrictions. In particular, the hierarchy extends only two more levels below this directory (e.g. crc/dps/handle). There is room for one more, so that when/if we achieve `.org' status we can add that level to the hierarchy.
