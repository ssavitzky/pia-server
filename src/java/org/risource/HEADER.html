This directory contains the Java reference implementation of the PIA.

Annotated Contents

content/
Representations for message content
dom/
An implementation of the W3C's Document Object Model
dps/
Document Processing System
ds/
Data Structures
pia/
The PIA proxy server itself
test/
Test cases and drivers (obsolete)
tf/
Classes that computer Transaction Features
util/
Utilities

Naming Conventions

Every effort has been made to reduce directory depth in order to accommodate Microsoft and ISO9660 restrictions. In particular, the hierarchy extends only two more levels below this directory (e.g. dps/handle).
