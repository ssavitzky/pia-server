Namespace Implementation Classes for the DPS

[..] [files]

This directory contains classes that implement the Namespace interface.

Namespace Classes

These are implementations of the Namespace interface.

BasicNamespace
Implements Namespace in the most straightforward way possible.
NamespaceWrap
Implements Namespace by ``wrapping'' an implementation of Tabular.
BasicEntityTable
Implements EntityTable

Auxiliary Classes

These are auxiliary classes used in the implementation of the Namespace classes.

EntityWrap
This is an entity used to wrap an arbitrary object.
