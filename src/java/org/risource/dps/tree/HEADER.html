DPS Active Node implementation classes

[..] [files]

This directory contains classes that are used in the DPS for representing active documents (parse trees). Classes implement both the DPS Active Document (cursorial) interfaces, and the org.w3c.dom Document Object Model (arboreal) interfaces.

It must be noted that, although these classes implement the interfaces specified in the DOM, in some cases they fail to implement the specified behavior.

Classes

Classes in this package have names that start with ``Tree'' .

TreeNode
this is the abstract base class for all if the Node implementations in this package.
