intended for small, ad-hoc tests. (not under CVS control)
Input Extensions:
.html
Intended for use with the HTML tagset
.if
Intended for use with the basic or other active tagset.
.txt
Intended for use with the bootstrap tagset.
Output Extensions:
.ok
Output for comparison, known (or hoped) to be correct.
.out
Output (temporary; not under CVS control)
.diff
Comparison of .ok and .out (temporary; not under CVS control)
.log
STDERR output (temporary; not under CVS control)
Note:
Test files should not contain CVS id's unless you want to manually check in the corresponding .ok files without keyword expansion.
