DPS Document Processors

[..] [files]

This directory contains implementations of the org.risource.dps.Processor and org.risource.dps.TopContext interfaces.

Base Classes

BasicProcessor
Generic implementation of org.risource.dps.Processor.
TopProcessor
Generic implementation of org.risource.dps.TopContext.

Processor Classes

ActiveDoc
TopContext for an active document (either a web page or a hook) in the PIA (as opposed to stand-alone).
BasicProcessor
The basic implementation of the Processor interface
TopProcessor
The basic implementation of the TopContext interface.
../Filter
Process a document as a stand-alone filter. This is not currently an implementation of Processor, but it probably ought to be.
