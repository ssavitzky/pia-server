DPS Active Node classes

[..] [files]

This directory contains classes that are used in the DPS for representing active documents (parse trees).

Eventually it will be used exclusively for interfaces and classes unique to the DPS. Interfaces and classes that simply extend or implement the DOM (org.w3c.dom) will be in org.risource.dom and org.risource.dom.tree.

Interfaces

Interfaces in this package have names that start with ``Active''. In general, they are also extensions of org.risource.dps.Active and its extension ActiveNode.

ActiveNode

Classes

Classes in this package have names that start with ``ParseTree'' if they are implementations of ActiveNode. The names of auxiliary classes are less consistent.
