This directory contains the all-new, all-singing, all-dancing, document site structuring subsystem.

Annotated Contents

Interfaces

Resource
A document or container resource accessible via a URL
Document
A document resource.
Root
The root of a resource tree.
Realizable
A document that can be ``realized:'' moved from a virtual location to a ``real'' one.

Concrete Classes

Subsite
An implementation of Resource and Realizable specifically for containers.
SiteDocument
An implementation of Document.
Site
An implementation of Root
Listing
A virtual Document that contains a listing of the contents of a Subsite.
FileResource
An implementation of Resource that takes all of its configuration information from an underlying file or directory.
FileDocument
An implementation of Document that takes all of its configuration information from an underlying file.

Base Classes

AbstractResource
Abstract base class for all implementations of Resource.
ConfiguredResource
Abstract base class for Subsite and SiteDocument.
