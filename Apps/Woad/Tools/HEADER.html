Tools for WOAD. This directory contains auxiliary pages and tagsets used internally to the WOAD system.

Annotated Contents

Tagsets

src-file.ts
Tagset for listing "generic" source code. Also handles .wl files, which are normally derived from source files either by ``lister'' programs, or by hand-annotation.
src-html.ts
Tagset for listing HTML files and such derivatives as .php and .shtml files.
src-wrapper.ts
Meant for inclusion: this contains only definition of the ``wrapper'' element <-document->.
src-xhtml.ts
Tagset for listing PIA .xh (xhtml) files. Adds the tags in basic.ts

Documents

src-listing.xh
Source-code directory-listing file.
word-listing.xh
Keyword directory-listing file.
word-listing.xh
Default (annotation) directory-listing file.
new-note.xh
``advanced'' form for creating new notes: allows such things as the filename (``label'') and author to be specified.
