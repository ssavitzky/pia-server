PIA / Contrib / rsv.ricoh.com

This directory is for software, especially configuration files, contributed by PIA users and developers at Ricoh Silicon Valley, <rsv.ricoh.com> (including the Ricoh California Research Center, <crc.ricoh.com>).

The contents of this directory are not an official part of the PIA distribution, but are included in the hope that they will make it more useful.

Each directory in this subtree corresponds to a user. Under each user, the ``pia/'' subdirectory contains the contents of the that user's PIA root directory (``~/.pia/'' on Unix systems) -- the name is changed to avoid having special characters in the names. Other configuration files, if the user chooses to put them here, are not renamed.
