This file contains the scripts and executables which are required for ordinary operation of the PIA and the DPS's process command. In particular, it contains the pia and process commands themselves.

Annotated Contents

autorun.inf
Automagically runs the PIA when a CD-ROM is inserted onto a DOS system.
pia
Unix/Linux shell script for running the PIA
pia.bat
DOS/Windows batch file (shell script) for running the PIA.
pia_wrapper
Unix/Linux wrapper shell script for locating standard directories before running a Java-based application such as the PIA.
piajdk.bat
DOS/Windows wrapper script for locating standard directories before running a Java-based application such as the PIA
process
Unix/Linux shell script for processing a file offline using the DPS.
