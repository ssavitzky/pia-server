Numeric Handler Demo

The numeric handler demo shows how to use the following numeric handler attributes: Sum, Difference, Product, Quotient, Remainder, Sort.

Sum Numbers Separated by Whitespace

Description: Adds the numbers: 10 20 30 40 9.5, and returns the result.

Input: <numeric sum>10 20 30 40 9.5</numeric>

Result: 109.5

Sum an HTML Encoded List of Numbers

Description: Adds the numbers: 100 200 300

Input:<ul><numeric sum><li>100 <li>200 <li>300</numeric></ul>

Result:
    600

Using the Difference Attribute

Description: Compute the difference between 14 and 17. This should return -3.

Input:<numeric difference> 14 17 </numeric>

Result: -3

Using the Product Attribute

Description: Compute the product of 9 and 9.

Input:<numeric product>9 9</numeric>

Result:81

Using the Quotient Attribute

 Description: Compute the quotient of 20 and 10.

Input: <numeric quotient>20 10</numeric>

Result: 2.0

Using Remainder Attribute

Description: Compute the remainder of 17 and 10.

Input:<numeric remainder>17 10</numeric>

Result:7

Sort a List of Numbers in Ascending Order

 Description: Sort 88 77 11 99 in ascending order.

Input:<numeric sort>88 77 11 99 </numeric>

Result:11778899

Sort a List of Numbers in Descending Order

Description: Sort 88 77 11 99 in descending order.

Input:<ul><numeric sort reverse><li>88<li>77<li>11<li>99</numeric></ul>

Result:
    99887711
