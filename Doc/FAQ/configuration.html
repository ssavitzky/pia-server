How do I

change configuration options for an agency
copy PIA_HOME/Config/pia.prop to PIA_ROOT/Config/pia.prop and edit as required.
change an existing document
copy PIA_HOME/Agents/AGENT/DOCUMENT to PIA_ROOT/Agents/AGENT/DOCUMENT and edit as required.
Note: PIA_HOME is usually ~/pia or /pia1/pia, PIA_HOME is usually ~/.pia
