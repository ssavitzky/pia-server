How do I

change configuration options for an agency
copy PIA_DIR/Config/pia.prop to PIA_HOME/Config/pia.prop and edit as appropriate.
change an existing InterForm
copy PIA_DIR/Agents/AGENT/INTERFORM.if to PIA_HOME/Agents/AGENT/INTERFORM.if and edit as appropriate.
Note: PIA_DIR is usually ~/pia or /pia1/pia, PIA_HOME is usually ~/.pia
