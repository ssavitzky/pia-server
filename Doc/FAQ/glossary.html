What does ___ mean?
PIA
Platform for Information Applications -- we are building the software infrastructure to create information applications, with Network Office Appliances being the main target.
However, the platform can also can be used for Personal Information Agencies, or Peripheral Information Agencies.
Agent
A persistent software object "with an attitude"
Actor
The code that interprets and implements an action specified by an InterForm tag.
InterForm
Interface Form -- extended html documents.
