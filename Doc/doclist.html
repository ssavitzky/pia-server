PIA Document List

The following is a list of the essential PIA documentation. It is best to read this documentation with the PIA running as many of the links to other documents and agent files will work only under those circumstances.

For Everyone

These documents are for all users of the PIA.

Document Description
Installation Guide The PIA Installation Guide describes how to install the Platform for Information Applications.
Introducing the PIA An introduction to the PIA and its use.
Agent Guide A descriptive list of the sample agents that ship with the PIA.

For Information Developers

These documents are for those who wish to create agents or customize the PIA.

Document Description
Authoring Agents An overview of agent creation.
Basic Tagset An introduction to the basic tagset available for use in XHTML documents.
Entities An introduction to the entities available for use in XHTML documents.

For Application Developers

These documents are for those who wish to extend the PIA.

Document Description
PIA Tagsets An overview of the PIA tagsets including their relationship to SGML DTDs.
PIA Internals Manual An overview of the internal workings of the PIA including the Application Program Inteface. Creating agents that extend the functionality of the core agency, fixing bugs in core technology.
