Get and Set Handler Demo

The get and set handlers demo shows how set a variable and subsequently extract its value.

Set a Variable, then Retrieve It

Description: Create a variable "today" and set it to the value "Friday". Retrieve this value using the get handler.

Input: <set name=today>Friday</set>

Input: <get name=today></get>

Result: Friday
