Protect Handler Demo

The protect handler demo shows how to protect marked up nodes from being expanded.

Protect a Marked Up Node

Description: This example shows the result of protecting emphasized text. HTML characters have been encoded so that they will not be expanded.

Input: <protect markup>Silicon Valley is <em>the</em> place to program.</protext>

Result: Silicon Valley is <em>the</em> place to program.

Expand a Marked Up Node

Description: If the result attribute is used, marked up nodes are expanded in the normal way.

Input: <protect result>Silicon Valley is <em>the</em> place to program.</protext>

Result: Silicon Valley is the place to program.
