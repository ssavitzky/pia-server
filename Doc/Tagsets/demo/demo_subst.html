Subst Handler Demo

The subst demo performs text substitution on node content.

Substitute a Pattern with Text

Description: Match on "10" and replace with "12".

Input: <subst match=10 result=12>Columbus Day is October 10, 1998</subst>

Result: Columbus Day is October 12, 1998
