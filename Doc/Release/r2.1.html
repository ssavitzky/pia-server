Release Notes for PIA Release 2.1

Release 2.1 is not out yet. This document is a place-holder. For the list of things that still have to be done between r2.0 and r2.1, see r2.1.to-do.html.

Release 2.1 is the first stable release of the PIA.

We suggest that people who wish to use the PIA to develop applications but are not interested in being on the cutting edge wait for 2.1, but we believe that there are unlikely to be any major changes between the current release (2.0.7) and Release 2.1.

Subscribe to PIA-announce@RiSource.org to be notified of new releases. Copies of the PIA can be obtained from our web site, http//www.RiSource.org/PIA.
