This directory contains working notes by Steve Savitzky <steve@rii.ricoh.com>. At various times it may include to-do lists, random ideas about the PIA and agents, design and implementation notes, rough drafts, and so on.

The contents of this directory are subject to change without notice, and are not guaranteed to have more than a tenuous relationship with the state of the PIA at any given time.

Annotated Contents

Directories

Design/
design and architectural notes on individual projects, plus some general notes on the architecture of the PIA, how to design PIA tagsets, and so on.
Archive/
Archive for notes of historical interest; i.e. notes relating to completed or abandoned projects.

Application Notes

Application notes are expected to be of use to web designers trying to build applications around the PIA.

applications.html
Possible applications for the PIA.
standards.html
The relationship between the PIA and various standards, especially XML, SGML, and DOM.
websites.html
Using the PIA offline to maintain a public website.

Working Notes

Working notes are primarily intended for developers working on the the PIA code base.

naming.html
Working notes on names, especially agent names, in the PIA.
porting.html
Notes on porting the PIA to other programming languages.
projects.html
Suggested projects. In some cases this goes rather deeply into the design, but in most cases the design notes have been moved to individual project files in the Design subdirectory.
roadmap.html
Roadmap for near-term projects. This is mainly at an architectural level.
