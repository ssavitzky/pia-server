This directory contains working notes by Steve Savitzky <steve@rsv.ricoh.com>. At various times it may include to-do lists, random ideas about the PIA and agents, design and implementation notes, rough drafts, and so on.

The contents of this directory are subject to change without notice, and are not guaranteed to have more than a tenuous relationship with the state of the PIA at any given time.
