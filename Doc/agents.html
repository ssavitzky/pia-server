Guide to PIA Agents

This document describes the sample agents that ship with the PIA. PIA agents fall into three categories:

  • Standard agents that are automatically installed when the PIA is set up.
  • Optional agents that ship with the PIA and can be installed at the user's discretion.
  • Contributed agents made available by the PIA developer community.

Standard Agents

The agents currently available are:

  • Admin
  • PIA and ROOT
  • Toolbar
  • History

Admin

The Admin agent is the main agent. It manages all the other agents that make up the PIA Agency.

PIA and ROOT

These two agents serve the files that are in the PIA_DIR and the PIA_HOME directories respectively. The PIA and ROOT agents are of type Document-Oriented File System (DOFS).

The Toolbar Agents

There are two Toolbar agents. The localfile agent adds a control menu to pages proxied by the PIA agency.

The remote agent adds a control menu to pages proxied through your agency

History Agent

The History agent tracks all web pages that you visit from any browser, allowing you to search those URLs by date. To make best use of the History agent, set your browsers to use your local Information Agency as your proxy (this setting is usually found under the Network options available under edit/preferences, options/network, or some similar menu. You need to specify both the host name and port--8888 by default-- where your agency is running.)

Once you set your Information Agency as your proxy, browse to a few external web pages, then visit the History agent's home page. You should see a link to today's date (e.g. My/History / 1997 / 08 /04) When you jump to this link you will see a list of the pages you visited.

Optional Agents

The PIA provides a simple form for installing agents. Look for it in Admin/install_agent. The optional agents currently shipping with the PIA include:

  • Swifty
  • HelloWorld
  • View
  • demo

Swifty

The Swifty agent allows you to easily customize your PIA home page. It offers the option of adding useful links, provides a custom page title and a decorative image

HelloWorld

The HelloWorld agent allows the user to choose a custom greeting. This agent provides a simple example of form use for PIA developers.

View

The View agent allows you to view documents after those document are processed with a tagset or script.

demo

This agent serves as both a demonstration and testbed for the PIA's interform markup language and its formatter/interpreter. Its home page provides links to tag use examples.

Contributed Agents

Agents contributed by the PIA developer community can be found in the directory Contrib/<host.name>/Agents Install these agents much as you would any of the optional agents provided by Ricoh.

Users can place a copy of their agent directories in Contrib/HOST.NAME/ and submit them to the PIA group for inclusion in the general release. Before attempting to install a contributed agent, copy its directory and files to USR_DIR/Agents or PIA_DIR/Agents.
