Platform for Information Applications

[..] [files] ( Agents, Config, Contrib, Doc, bin, lib, src )

This is the top-level directory for software and documentation generated by members of the PIA Project at Ricoh CRC.

Annotated Contents

Directories

Agents/
Agent InterForms (active documents).
Config/
Configuration files for various PIA-based appliances.
Contrib/
Contributed software.
Doc/
``pure'' (non-active) documentation, including manuals, how-to guides, etc. to be included with the release.
bin/
Binaries (executable files). In addition to the PIA, this directory includes some additional applications here for use on WebWorts and other PIA-based appliances.
lib/
Libraries.
src/
Source code for the PIA and related utilities.

Naming Conventions

Sub-directories that contain source code, libraries, and executables follow the Unix naming convention: src, lib, and bin respectively. Subdirectories that contain documents (active or otherwise) have capitalized filenames: Doc, Agents, etc.
